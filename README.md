# Mini API de Tareas

API REST desarrollada con NestJS y PostgreSQL. (Acepta archivos adjuntos)

## Tecnologías

- NestJS
- PostgreSQL
- TypeORM
- Multer (archivos adjuntos)

## Requisitos

- Node.js >= 18
- PostgreSQL

## Instalación

npm install

## Configuración

Abre src/app.module.ts y actualiza los datos de conexión a PostgreSQL:

- host
- port
- username
- password
- database

## Correr el proyecto

npm run start

## Endpoints

Todos los endpoints requieren el header: authorization: token-secreto-sorah

## Uso de IA

Consulté a Claude para resolver dudas que tuve con TypeORM y algunos aspectos de estructura.
